Design principles

  • Separation of concerns — audio/transcription logic stays in VoiceToText; agent logic will live in separate modules.
  • Incremental build — start with a working transcription path, then layer agents on top of stable text output.
  • Living documentation — root-level docs are updated whenever the codebase changes (see sync-root-docs.mds).
Last aligned with initial project scaffold — update when new modules or integrations are added.
